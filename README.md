# 阅读越有fun - 微信小程序与后台管理系统

一个帮助用户组织和参与共读活动的微信小程序，让阅读变得更有乐趣和社交性。包含微信小程序前端、Node.js后端API和Vue3管理后台系统。

## 项目结构

项目由三个主要部分组成：

1. **Front-end/** - 微信小程序前端
2. **Back-end/** - Node.js + Express 后端API
3. **Admin-system/** - Vue3 + Element Plus 管理后台

## 本地部署指南

### 后端部署

1. 安装依赖:
   ```bash
   cd Back-end
   npm install
   ```

2. 配置环境变量:
   创建或修改 `.env` 文件，添加以下内容:
   ```
   PORT=3000
   MONGODB_URI=mongodb://localhost:27017/reading_fun_dev
   JWT_SECRET=your_jwt_secret
   ADMIN_JWT_SECRET=your_admin_jwt_secret
   WECHAT_APPID=your_wechat_appid
   WECHAT_SECRET=your_wechat_secret
   ```

3. 启动开发服务器:
   ```bash
   npm run dev
   ```
   
4. 初始化管理员账户:
   ```bash
   npm run create-admin
   ```

### 管理后台部署

1. 安装依赖:
   ```bash
   cd Admin-system/client
   npm install
   ```

2. 配置API路径:
   修改 `vite.config.js` 中的代理配置，确保指向正确的后端API地址

3. 启动开发服务器:
   ```bash
   npm run dev
   ```

4. 构建生产版本:
   ```bash
   npm run build
   ```

### 微信小程序部署

1. 使用微信开发者工具打开 `Front-end` 目录
2. 配置项目AppID（可使用测试号）
3. 修改 `services/api.js` 中的API基础URL，指向你的后端服务地址
4. 预览或真机调试
